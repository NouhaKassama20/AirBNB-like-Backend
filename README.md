# AirBNB-like-Backend
1. git clone <url>
2. npm install
3. Créer .env avec les bonnes valeurs

.env: 
PORT=5000
MONGO_URI=mongodb://localhost:27017/airbnb_algeria
JWT_SECRET=airbnb_algeria_secret_2024_xyz
JWT_EXPIRE=7d
NODE_ENV=development
CLIENT_URL=http://localhost:5173

.env (frontend):
VITE_API_URL=http://localhost:5000/api
